# git-practice
git basic practice


1. github fork 老师准备的repository，假设inblackhole是老师账户
github操作fork https://github.com/inblackhole/git-practice/ 到自己的github

3. git clone
在电脑本地，clone 上面已经fork的自己github账户下的repository

4. github Sync fork
clone后，被fork的老师的repository如果有新的变化，在github上操作sync fork来同步变更

6. git pull
上一步同步的变更仍然在github上，需要git pull拉取到本地
 
8. add and edit a file
基于最新的代码，随意加一个文件，比如hello.py，里面代码随意

10. git add
把这个新文件加到git管理中

12. git commit
提交变更到git

14. git push
把变更反应(推送)到github

16. github pull request
github操作pull request，向老师申请PR
